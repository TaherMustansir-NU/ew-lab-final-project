C #1 COURTESY

Be respectful, tactful, and professional.

✓
Use sincere and thoughtful language.
✓
Choose expressions that show respect.
✓
Avoid harsh or biased wording.
✓
Critique ideas, not people.
✓
Close with a constructive tone.
Wrong
"You did this wrong. Fix it now."
Correct
"There is an issue in this section. Could you please revise it today?"
🤝 Quick Focus

Use calm and respectful language even when the message is corrective.

  • Start polite, stay clear.
  • Request changes, do not attack people.
  • End with thanks or support.

C #2 CONSIDERATION

Focus on the receiver, not just the sender.

✓
Use a "you" orientation where suitable.
✓
Highlight audience benefit early.
✓
Show empathy for reader constraints.
✓
Adjust tone and detail to the audience.
✓
Present information in an easy-to-use format.
Wrong
"I need this because my department requires it."
Correct
"If you send this today, your request will be approved before weekend closure."
👥 Quick Focus

Frame the message around what the receiver needs to know and do next.

  • Lead with audience benefit.
  • Match tone to context.
  • Keep instructions easy to follow.

C #3 CONCISENESS

Say more with fewer, stronger words.

✓
Eliminate wordy expressions.
✓
Include only relevant material.
✓
Avoid unnecessary repetition.
✓
Use short direct sentences.
✓
Cut words that add no meaning.
Wrong
"At this point in time, we are in a position to begin."
Correct
"We can begin now."
✂️ Quick Focus

Trim every line until only the useful message remains.

  • Delete repeated words.
  • Prefer short verbs.
  • Keep one point per sentence.

C #4 CLARITY

Make meaning obvious on first read.

✓
Choose precise, familiar words.
✓
Arrange points in a logical order.
✓
Use one main idea per sentence.
✓
Use transitions to connect ideas.
✓
Remove words that create ambiguity.
Wrong
"They should review it before submission."
Correct
"Team leads should review the report before Friday submission."
🧭 Quick Focus

If a sentence can be read two ways, rewrite it.

  • Name roles clearly.
  • Use specific deadlines.
  • Avoid vague pronouns.

C #5 COMPLETENESS

Give the reader everything needed to act.

✓
Provide all necessary information.
✓
Answer who, what, when, where, why, how.
✓
Add detail where it prevents confusion.
✓
State action and deadline clearly.
✓
Include context or evidence for claims.
Wrong
"Please send the report soon."
Correct
"Please send the Q1 sales report by 3:00 PM Friday to finance@company.com."
📋 Quick Focus

Before sending, check whether the reader has enough detail to act once.

  • What is needed?
  • Who does it and by when?
  • Where to send/submit?

C #6 CONCRETENESS

Replace vague language with specific proof.

✓
Use specific facts and figures.
✓
Use active, precise verbs.
✓
Choose vivid words that create meaning.
✓
Replace broad claims with measurable outcomes.
✓
Use real examples, not abstract claims.
Wrong
"Sales improved a lot this month."
Correct
"Sales increased 18% in March after the referral campaign."
📈 Quick Focus

Replace broad claims with numbers, names, and concrete results.

  • Use percentages or counts.
  • Mention specific actions.
  • Include one real example.

C #7 CORRECTNESS

Be accurate in language, facts, and format.

✓
Use the right level of language for your audience.
✓
Check grammar, spelling, and punctuation.
✓
Verify figures, facts, names, and dates.
✓
Maintain consistent formatting and mechanics.
✓
Review once more before sending or publishing.
Wrong
"Their are 35 employes in the branch, data was checked yesterday."
Correct
"There are 35 employees in the branch, and the data was checked yesterday."
✅ Quick Focus

Accuracy is the final quality filter before delivery.

  • Proofread grammar and spelling.
  • Verify all facts and names.
  • Keep formatting consistent.
